# L'ÉCHO DU MATIN 🗞️

Application d'actualités stratégiques propulsée par Gemini 3.
Propriété exclusive de **Atmani Bachir**.

## 🚀 DÉPLOIEMENT SUR VERCEL

### Étape 1 : Préparation
- Créez un nouveau projet sur Vercel.
- Liez votre dépôt GitHub/GitLab.

### Étape 2 : Configuration Cruciale (Variable d'Environnement)
Dans les paramètres de Vercel (**Settings > Environment Variables**), vous devez impérativement ajouter :
- **Name :** `API_KEY`
- **Value :** Votre clé Google Gemini (`AIzaSy...`)

### Étape 3 : Build
L'application utilise Vite. Vercel détectera automatiquement les réglages. Cliquez sur **Deploy**.
